# Шаблоны настроек

Это шаблоны настроек для https://hub.docker.com/r/wiktorbgu/mihomo-mikrotik

## Как использовать

Обычно шаблоны по SFTP кладут в папку роутера /usb1/docker_configs/mihomo_mikrotik/template

## Как настраивать

В переменные ENV окружения вашего контейнера. Отличие шаблонов можно почитать внутри.

### Шаблон большой 

Если вы хотите все и сразу, но переключателей много, можете запутаться. Потребляет много памяти, если раскоментировать отключенные функции - потребит еще больше памяти, около 40МБ и больше.

`CONFIG="stargate.yaml"`

```
/container envs add list=MIHOMO key=CONFIG value="stargate.yaml"
```

### Шаблон облегченный

Для домашнего использования подходит лучше всего. Потребляет около 20МБ памяти.

`CONFIG="stargate-lite.yaml"`

```
/container envs add list=MIHOMO key=CONFIG value="stargate-lite.yaml"
```

### Принцип настройки для первого запуска

Подходит для обоих шаблонов

* переключите выключатель SELECTOR на желаемый режим работы
* установите значение PROXY MANUAL если нужен какойто определенный прокси для серисов
* выберите уже нужные режимы или сервера под конкретные сервисы как угодно
  
Комментарии в файле есть

### Важное замечание  

Если потрогаете Fallback/URLtest выключатели, то автоматическая логика ломается, прикасайтесь только к выключателям типа Selector.

Чинится тестом скорости - правой кнопкой мыши на названии переключателя.
